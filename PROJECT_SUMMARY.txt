HOW TO USE
================================================================================

METHOD 1: Direct File Opening
------------------------------
1. Navigate to: /Users/juanminni/starken-financeiro/
2. Double-click index.html
3. Use sidebar navigation to access all pages

METHOD 2: Local Development Server (RECOMMENDED)
------------------------------------------------
1. Open terminal in project directory
2. Install dependencies (first time only):
   $ npm install
3. Start the server:
   $ npm start
4. Open browser and navigate to:
   http://localhost:3000

The server will show all available page URLs.
